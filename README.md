## Flask Service
### For microservice purpose
### Step installation and run your server 
1. Setup virtual environment
    ```shel
    python -m venv venv
    ```
2. Use virtual environment
    ```shell
    source venv/bin/activate
    ```
3. Install requirements
   ```shell
   pip install -r requirements.txt
   ```
4. Run http app
    ```shell
    python run.py http
    ```
